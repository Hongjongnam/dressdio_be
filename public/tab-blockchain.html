      ⚡ 5. Dress Token 전송 + 자동 스왑 (플랫폼 전용) (POST

🚀 원클릭 통합 스왝!

      

        한 번의 클릭으로 Dress Token 전송과 DP Token 스왑을
        모두 처리합니다!

        받는 주소 입력 불필요 - 자동으로 플랫폼 어드민 지갑으로
        전송됩니다.
      

    


    

      
🔄 자동 프로세스:

      
    
        
  1. 
          Dress Token 전송 (Polygon) → 플랫폼 어드민 지갑
        
    2. 
        
  2. 트랜잭션 검증 → Transfer 이벤트 파싱
    3. 
        
  3. 
          DP Token 자동 지급 (Besu) → DRESS × 5배 (1:5 비율)
        
    4. 
      

      

        💡 예시: 100 DRESS 입력 → 자동으로 500 DP 받기!
      

    


    

      
      

      
      

      

        

          ⚠️ 주의: 받는 주소는 자동으로 플랫폼 어드민 지갑

      6. Dress → DP Token Swap (수동) (POST /api/utils/swap-dress-to-dp)
    
    

      💡 Tip: API #5에서 "전송 + 자동 스왑"을 사용하면 더
      편리합니다!

      여기는 이미 전송한 트랜잭션을 수동으로 스왑할 때만
      사용하세요.

  1. API #1로 DRESS & MATIC 잔액 확인
    2.

          0x2C686C...312871로 전송
        
    3. 
        
  3. 전송 완료 후 트랜잭션 해시 복사
    4.
