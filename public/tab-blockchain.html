    5. Dress Token 전송 + 자동 스왑 (POST /api/utils/dress-token/transfer-and-swap)
    

      한 번의 요청으로 Dress Token 전송과 DP Token 스왑을 모두 처리합니다.
      받는 주소 입력 불필요 — 자동으로 플랫폼 어드민 지갑으로 전송됩니다. (1 DRESS = 5 DP)
    

    

      
      

      
      
      입력한 금액의 5배 DP를 받습니다. 받는 주소는 자동으로 플랫폼 지갑(0x4c7e5b...4674db)으로 설정됩니다.
