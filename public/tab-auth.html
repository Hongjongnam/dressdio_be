📧 이메일/비밀번호 인증

이메일 회원가입

이메일과 비밀번호로 새 계정을 생성합니다. 이메일 인증 코드가 필요합니다. 


        


        
        

           이메일 로그인
          
이메일과 비밀번호로 로그인합니다.

          

            
            

            
            

            
          

          

        


        
        

           이메일 존재 확인
          

            이메일 주소가 이미 등록되어 있는지 확인합니다. 회원가입 전에
            확인해보세요.
          

          

            
            

            
          

          

        


        
        

           이메일 인증 코드 발송
          

            회원가입에 필요한 이메일 인증 코드를 발송합니다.
          

          

            
            

            
            

            
            

            
          

          

        


        
        

          
             이메일 인증 코드 확인
          
          
발송된 이메일 인증 코드를 확인합니다.

          

            
            

            
            

            
          

          

        


        
        

           비밀번호 재설정
          

            이메일 인증 코드를 사용하여 비밀번호를 재설정합니다.
          

          

            
            

            
            

            
            

            
          

          

        

      


      
      

        
🔗 소셜 로그인


        
        

          
             1. 소셜 로그인 URL 요청
          
          

            소셜 로그인 제공자를 선택하여 인증 URL을 받습니다. 사용자는 이 URL로
            이동하여 로그인합니다.
          

          

            
            
            
            
          

          

        


        
        

           2. 로그인 완료
          

            콜백 URL의 ID를 입력하여 로그인을 완료하고 액세스 토큰을
            발급받습니다.
          

          

            
            
            
          

          

        


        
        

           3. MPC 지갑 생성 / 복구
          

            2단계에서 받은 액세스 토큰과 장치 비밀번호를 입력하여 MPC 지갑을
            생성하거나 복구합니다.
          

          

            
            
            
            
            
            
            
          

          

        


        
        

           계정 정보
          

            액세스 토큰을 수동으로 입력하여 계정 정보와 토큰 잔액을 조회합니다.
          

          
          
          

            

              
            

            

              
            

          

          

          

        


        
        

           토큰 재발급
          

            액세스 토큰이 만료된 경우, 리프레시 토큰을 사용하여 새로
            발급받습니다.
          

          

            
            
            
          

          

        


        
        

           소셜 회원가입
          

            로그인 완료 시 618 에러(신규 사용자)가 발생한 경우, 이 양식을
            사용하여 가입을 완료합니다. 에러 응답의 `email`과 `txId`를
            입력하세요.
